# AI Projects 板块设计升级说明

## 设计理念

作为高级 UI 设计师，本次升级遵循以下核心设计原则：

### 1. 视觉层次优化
- **大号装饰性数字**：项目编号使用 6xl 大小，10% 透明度，作为背景装饰元素
- **标题增强**：标题字号提升至 3xl/4xl，增强视觉冲击力
- **间距优化**：增加各元素间距（mb-6, mb-8），提升呼吸感

### 2. 色彩和谐设计

#### Art Mode（浅黄背景）
- **卡片背景**：80% 透明度白色 + 毛玻璃效果（backdrop-blur-sm）
- **悬停效果**：2xl 阴影 + 1.02 倍缩放，营造浮起感
- **渐变装饰**：primary/3 到 accent/5 的微妙渐变
- **数据卡片**：accent/30 到 accent/10 的渐变背景

#### Code Mode（纯黑背景）
- **卡片背景**：50% 透明度深灰 + 毛玻璃效果
- **边框光效**：primary/20 边框 + 微妙的横向光效
- **悬停效果**：primary/50 边框 + 80% 透明度背景
- **标题渐变**：foreground 到 foreground/70 的文字渐变

### 3. 微交互动画

#### 图标动画
- **旋转效果**：悬停时 360° 旋转 + 1.1 倍缩放
- **背景容器**：primary/10 背景 + primary/20 边框（Code Mode）

#### 技术标签动画
- **入场动画**：逐个淡入 + 缩放（延迟 0.05s）
- **悬停效果**：
  - Code Mode：边框变亮 + primary/10 背景
  - Art Mode：accent 背景 + 边框变化

#### 按钮动画
- **悬停效果**：1.05 倍缩放 + 向上移动 2px
- **点击效果**：0.95 倍缩放
- **Code Mode**：primary/10 背景 + primary/20 悬停
- **Art Mode**：渐变背景 + 阴影增强

### 4. 装饰元素

#### 卡片装饰
- **渐变背景层**：悬停时显示，从 primary 到透明的微妙渐变
- **边框光效**：Code Mode 专属，横向扫光效果
- **毛玻璃效果**：backdrop-blur-sm，增强层次感

#### 标题区域装饰
- **小标题徽章**：accent/20 背景 + 边框，居中显示
  - Art Mode：✦ 精选作品 ✦
  - Code Mode：// Featured Projects
- **装饰性下划线**：渐变横线，从透明到 primary 再到透明

### 5. 排版优化

#### 文字层级
- **主标题**：5xl/6xl，加粗
- **项目标题**：3xl/4xl，加粗
- **描述文字**：base/lg，行高 relaxed
- **标签文字**：xs/sm，适度间距

#### 间距系统
- **卡片内边距**：p-8/p-10（增加舒适度）
- **元素间距**：mb-6, mb-8（统一节奏）
- **卡片间距**：gap-12（增强分隔感）

### 6. 数据展示优化

#### Code Mode
- **JSON 风格**：muted/50 背景 + 边框 + 内阴影
- **逐行动画**：每行延迟 0.1s 淡入
- **字体加粗**：primary 颜色的 key 使用 semibold

#### Art Mode
- **渐变卡片**：accent/30 到 accent/10 的渐变背景
- **大号数字**：3xl 字号，加粗
- **逐个动画**：每个卡片延迟 0.1s 淡入 + 上移

### 7. 获奖徽章优化
- **渐变背景**：amber-500/10 到 orange-500/10
- **边框增强**：amber-500/30
- **入场动画**：缩放 + 淡入（延迟 0.3s）

### 8. 项目链接按钮优化

#### Code Mode
- **背景**：primary/10 + primary/20 悬停
- **边框**：primary/30
- **字体**：等宽字体

#### Art Mode
- **渐变背景**：accent 到 accent/80
- **阴影**：md 到 lg 的阴影过渡
- **悬停效果**：渐变色变化

## 设计亮点

### ✨ 与背景和谐融洽
- Art Mode：透明白色卡片与浅黄背景完美融合
- Code Mode：半透明深色卡片与纯黑背景形成对比

### ✨ 高级感营造
- 毛玻璃效果（backdrop-blur）
- 微妙的渐变装饰
- 精致的悬停动画
- 专业的排版节奏

### ✨ 双模式适配
- 每个元素都有 Art/Code 两种样式
- 颜色、字体、动画都针对性优化
- 保持统一的设计语言

### ✨ 细节打磨
- 装饰性大号数字
- 图标旋转动画
- 标签逐个入场
- 数据卡片动画
- 按钮微交互

## 技术实现

### 关键技术
- **Framer Motion**：所有动画效果
- **Tailwind CSS**：响应式布局和样式
- **CSS 渐变**：背景装饰和按钮效果
- **毛玻璃效果**：backdrop-blur-sm
- **透明度控制**：精确的透明度层级

### 性能优化
- 使用 CSS transform 而非 position 动画
- pointer-events-none 避免装饰层干扰交互
- 合理使用 transition-duration
- viewport 优化，只在可见时触发动画
